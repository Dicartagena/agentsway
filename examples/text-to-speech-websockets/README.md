# End to end websockets example

1. cd into `node-express`
2. run `npm install`
3. copy `.env.example` to `.env` and add in your ElevenLabs key
4. run `npm start`
5. open a new terminal
6. cd into `nextjs`
7. run `npm install`
8. run `npm start`
9. open up `localhost:3000` and type in a sentence then hit generate
